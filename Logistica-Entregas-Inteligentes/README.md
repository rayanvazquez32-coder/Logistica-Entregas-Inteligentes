# Logística de Entregas Inteligentes

## Descripción

Aplicación desarrollada en Python para administrar clientes, paquetes y repartidores. El sistema utiliza Programación Orientada a Objetos, persistencia de datos mediante archivos CSV y teoría de grafos para calcular rutas de entrega entre diferentes zonas de la ciudad.

## Funcionalidades

* Registro de clientes.
* Registro de paquetes.
* Gestión de repartidores.
* Cambio de estado de paquetes.
* Búsquedas rápidas mediante diccionarios.
* Guardado y carga automática de datos.
* Cálculo de rutas mediante grafos y búsqueda DFS.
* Generación de reportes ordenados por peso, distancia o fecha de entrega.
